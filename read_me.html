Always Make a Backup!

If you upload Helios files to your website without making a backup there is no way to resolve any problems that may arise. You should always make a backup of your website files and database before installing or upgrading any application.

The following instructions are an overview for experienced users. If you need more detailed instructions please visit the documentation.

Installation

Note: If your site already has any of the three files below do not upload the corresponding Helios files without renaming them. If you upload the entire zip you may overwrite your existing files which could break your site.

root index.php
/events directory
/admin directory

Optionally you can install Helios in it's own subdirectory. /calendar is a popular choice.
  1. Upload the contents of the Helios ZIP to your web server.
  2. Create Your Helios database & database user account
    (user must have create & alter permissions)
  3. Rename globals_dist.php to globals.php and edit it to use your server settings.
  4. Run the Helios web setup (You will be prompted for this by opening your Helios Calendar URL in your browser)
Step by Step instructions are available in the Helios Documentation.


Upgrading

Note: Do not upload the following files or you will overwrite your existing Helios template and revert any customizations you've made.

/events/index.php
/events/css/helios.css
/events/template/*.*

Important: Do not delete, or overwrite your existing globals.php. You will not need to alter it (unless instructed to do so at the link below) as part of the upgrade process.

Preparation: If you are using a version of Helios older then 1.3.1 you must run the upgrade131.php file before running the upgrade.php file. Follow the same steps as in #2-#4 below but with the upgrade131.php file, then repeat with the upgrade.php file.
  1. Upload the contents of the Helios Zip (all except the /events/setup directory) to your web server.
  2. Upload /events/setup/upgrade.php to /events/upgrade.php
  3. Point your browser to YourHeliosInstall/events/upgrade.php and run the appropriate upgrade.
  4. Delete the upgrade.php file, and the contents of your /events/cache/ directory (keep the directory), from your web server.
Important: Depending on the version you are using at the time of upgrade you may need to make some additions or changes manually.

For additional, version specific, upgrade instructions please visit the documentation.
