Installation Instructions

1) Upload the contents of the Helios ZIP to your webserver.
2) Create Your Helios database & database user account (user must have create permissions)
3) Configure your globals.php file (/events/includes/globals.php)
4) CHMOD 666 your WURFL log file (/events/includes/wurfl/data/wurfl.log)
5) Run the Helios web setup (You will be prompted for this by trying to open Helios in your browser)
